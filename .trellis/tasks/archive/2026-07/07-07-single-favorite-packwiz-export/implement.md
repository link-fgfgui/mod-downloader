## Risk Points

- Wails generated bindings must be refreshed after adding exported Go methods.
- Provider version resolution can require network/API availability when cache is cold.
- CurseForge direct download URLs may require API key behavior in some cases; packwiz consumers may not have this app's API key.
- Loader version is not currently exposed in `VersionInfo`; adding it touches launcher parsing and frontend models.
- Favorite list display is not currently partitioned by Minecraft version + modloader tuple; this task intentionally fails mixed-scope exports instead of implementing that UI/storage TODO.
- TOML output should be generated by structured encoding or carefully tested rendering, not ad hoc untested string concatenation.

## Rollback Plan

- Keep the exporter isolated so UI and Wails additions can be reverted without changing favorite storage.
- Avoid database migrations for this task.
- Write the ZIP through a temp path and remove it on failure.
